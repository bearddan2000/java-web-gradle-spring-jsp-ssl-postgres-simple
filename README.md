# java-web-gradle-spring-jsp-ssl-postgres-simple

## Description
A jsp springboot java gradle build,
that connects to self-signed ssl enabled postgres database.

## Tech stack
- springboot
  - web
- jsp
- gradle
- bootstrap
- jquery
- dataTable

## Docker stack
- alpine
- postgres
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
